# Предложенные редакторские правки

Оригинальные главы не тронуты. В этой папке лежат копии с правками для ревью.

## Что проверяем этим проходом

- Сокращение поясняющих хвостов там, где сцена уже работает сама.
- Меньше общих формул про взрослых; больше конкретного поведения персонажей.
- Чуть больше участия Джима в раскрытии тайны в 6-й главе.
- Более двусмысленная подготовка Сильвера в 7-й главе: тревога остается, но он не выглядит заранее подсвеченным злодеем.

## Основные точки

- `chapter-01-draft.md`: точечная чистка повторяющихся обобщений и финального объяснения страха перед одноногим.
- `chapter-02-draft.md`: Ливси меньше объясняется через авторский комментарий, больше через действие в сцене.
- `chapter-03-draft.md`: финальная мысль после смерти Бонса сжата, чтобы карточка с черным кругом работала жестче.
- `chapter-04-draft.md`: папка под курткой описана физически, без прямого "я стал другим человеком".
- `chapter-05-draft.md`: сцены за баками и смерть Пью оставлены более предметными, меньше общих выводов.
- `chapter-06-draft.md`: раскрытие бумаг Бонса собрано через вырезки, карту, фото и наблюдение Джима про кольцо; убран дублирующий поздний опрос по фотографиям.
- `chapter-07-draft.md`: письмо Трелони и разговор про Сильвера сделаны менее прямолинейными; подозрение остается через поведение Трелони и формулировки письма.
